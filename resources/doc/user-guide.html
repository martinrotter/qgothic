QGothic is simple board game developed using Qt framework. It is based on a well-known Altdeutsche Dame.

General

QGothic is simple checkers-like board game. It can be played in a very similar way to traditional chess, despite little different game rules. This game is based on Damn-Spel and Czech Checkers. Altdeutsche Dame has been inferred because of the similarities between south German and Turkish checker rules that Altdeutsche Dame may have influenced Turkish Checkers, invented around 1683.

Game Rules

Note that QGothic's rules are different from traditional checkers.

Beginning of a Game

  1. Usually white player starts, but sometimes this can be re-negotiated by players, so that black player can start sometimes too.

Course of a Game

  1. Game is played on rectangular board of 8x8 size.
  2. Number of player is strictly limited to 2 players.
  3. Players change in their turns regularly, one move is made during one turn.
  4. Each player starts with 12 pawns located in 2 southest rows from view of that respective player.
  5. Pawn can move diagonally by one field forward. Moving backwards is not allowed.
  6. Pawn can jump opponents's figures. Jumped figure has to be located just next to jumping pawn and field behind victim must be empty.
  7. These simple pawn jumps can be chained in a jumping sequence.
  8. If pawn reaches furthest row of board, it is promoted to the queen and move is ended. It's opponent's turn now.
  9. Queen is allowed to move diagonally to all directions.
  10. Queen can jump figure in its way, if there is at least one field empty behind jumped figure.
  11. These simple queen jumps can be chained in a jumping sequence.
  12. Each player must make a move containing the most available figures. Otherwise any move can be made.
  13. If player is not able to make any move, then it's opponent's turn.

Ending of a Game

  1. At least one player has no figures remaining.
  2. There have been made certain number of non-jumping moves. This number is usually 30 moves per player. Winner is the player with more figures in this case. Same number of figures for both player means, that game ended in a draw. Points are given for each remaining figure in this state sometimes. Queens are evaluated by 4 points and pawns by 1 point. Points are summed up for each player and player with more points win. Same number of points means draw again.

Starting/Saving/Loading Games

Starting Game

Game can be started by triggering Game -> New action or hitting CTRL+N key shortcut. User is asked to save his current game first. If new game is started, then beginning player is loaded from settings. So changing beginning player in a middle of active game has no signification for that actual game, but for newly started games only. But type of both players can be changed in-game and apply for currently active game.

Loading Game

Game can be loaded by triggering Game -> Load action or hitting CTRL+L key shortcut. User is asked to save his current game first. QGothic supports its internal *.qgo files only. If loading fails, it is announced to the user.

QGothic save game files store variety of things.
  • entire history of moves
  • difficulty and type of both players
  • maximal allowed number of jumped figures
  • color of beginning player
  • result of the game

Game is redo-ed to its last move, when it's loaded. So if game contains eg. 50 moves, it's rolled "forward" to the last player move.

NOTE: Games saved under X.Y.Z build cannot be loaded in other versions of QGothic.

NOTE: QGothic supports opening saved games from your desktop directly by clicking the saved game and dragging saved games onto QGothic main window too.

Saving Game

Game can be saved by triggering Game -> Save action or hitting CTRL+S key shortcut. Also read the above note.

Configuration

Configuration dialog can be accessed by firing Tools -> Configuration action or hitting CTRL+C key shortcut. Configuration dialog allows you to edit some things like:
  • type and difficulty of both players
  • beginning player etc.
  • appearance of application

Miscellaneous

Graphical Board

Board allows you to view moveable figures which can be highlighted with color when one of them is clicked. Also opponent's moveable figures can be highlighted as well by clicking on any of opponent's figure.

Pausing and Resuming Game

Game mustn't be paused, if human player wants to make his turn. If AI has his turn, game can be paused, but sometimes pausing of game can take several seconds.
