int Strategy::simple ( Board &  board)
static

Very simple board-evaluating functions.

This functions prefers statical portion of board rating. That makes AI with this strategy rather defensive.

Parameters
boardBoard to rate.
Warning
Rating just statical part of the board could make this strategy "blind" as sometimes count of figures is not everything. But it does basic job pretty well.
Returns
Method returns rating of board from the view of active player. The higher integer the better position for active player.
