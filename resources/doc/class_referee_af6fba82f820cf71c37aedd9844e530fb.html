QList< Move > Referee::getBestMoves ( const QList< Move > &  moves)
static

Best moves selector.

Method selects longest available moves from input list. Longest move means move with the highest count of jumped figures. If more moves have the same highest count of jumps, then all figures are included in result.

Parameters
movesMoves to filter.
Returns
Method returns filtered list of longest moves.
